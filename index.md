## What You Need

### Hardware

* An **autopiper** board.
* **USB cable** - USB A / micro USB B.
* **Computer** running Windows, Linux, or macOS.



### Software

To start using ESP-IDF on **autopiper**, install the following software:

> * **Toolchain** to compile
> * **Build tools** 
> * **Autobuilder**
